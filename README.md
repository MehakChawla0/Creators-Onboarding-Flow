# CreatorsMela – Creator Onboarding Flow

## Overview

This submission is a high-fidelity mobile onboarding experience designed for **CreatorsMela**, an Indian creator marketplace that connects content creators with brands for collaboration opportunities.

The goal of the redesign was to create a frictionless, intuitive, and visually engaging onboarding journey that helps creators set up their profile quickly while building trust and reducing drop-offs.

---

## Design Goals

* Reduce onboarding friction
* Guide users step-by-step
* Improve profile completion
* Build trust through transparent verification
* Create a premium first-time user experience
* Maintain a clean, modern, and accessible interface

---

## Key Features

* Welcome & Account Setup
* Profile Creation
* Niche Selection
* Social Account Verification
* Audience Insights
* Creator Rate Card
* Profile Review
* Success & Completion Screen
* Dark Mode Dashboard
* Progress Indicators
* Micro-interactions & Smooth Navigation
